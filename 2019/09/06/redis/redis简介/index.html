redis 简介

redis 是一个高性能的 key - value 数据库。

特点

  • 支持数据持久化，内存-》磁盘-》内存
  • 丰富的数据结构（简单 key-value， list， set， zset， hash 等）
  • 数据备份，master-slave模式的数据备份

    优势

  • 高性能
  • 丰富的数据类型
  • 原子性
  • 丰富的特性（publish/subcribe, key 过期， 通知等等）

    可用处

  • 排行榜（SortSet）
  • 计算器/限速器（用户点赞，访问量），（用户访问 API 频率）
  • 好友关系（并集，差集，交集，共同好友，共同爱好）
  • 简单消息队列（list 队列机制，到货通知，邮件发送）
  • Session 共享

    不能干

    在数据量很大，数据访问频率很低的场景不适合使用

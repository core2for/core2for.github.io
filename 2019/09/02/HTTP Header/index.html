HTTP Header

  • Pragma: no-cache 强制要求缓存服务器在返回缓存的版本之前将请求提交到源头服务器进行验证。
  • Cache-control: no-cache… 实现缓存机制（单向）
  • Expires: Wed, 21 Oct 2015 07:28:00 GMT 在此时候之后，响应过期。
    无效的日期，比如 0, 代表着过去的日期，即该资源已经过期
